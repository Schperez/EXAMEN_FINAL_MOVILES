  - Introducción del proyecto

Este proyecto consiste en el desarrollo de un sistema de cámara inteligente capaz de realizar seguimiento de rostro mediante visión artificial y control de movimiento con servomotor. El sistema integra una aplicación móvil desarrollada en Kotlin, comunicación Bluetooth, Arduino y procesamiento de imagen con Python y OpenCV.

La aplicación permite controlar manualmente la posición del servo desde un teléfono Android, conectándose al módulo Bluetooth del sistema. Además, cuenta con un modo automático en el que el control manual se bloquea y el movimiento queda a cargo del programa de visión artificial, el cual detecta el rostro y envía comandos al Arduino para mantenerlo centrado.

El objetivo principal del proyecto es demostrar la integración entre software móvil, electrónica, comunicación inalámbrica y visión computacional, aplicándolo a una cámara asistente para videollamadas, presentaciones o sistemas de monitoreo inteligente.
